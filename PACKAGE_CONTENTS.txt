🚀 快速开始
----------------------------------------

1. 解压文件:
   tar -xzf english_grammar_tutor_v2_780words.tar.gz

2. 进入目录:
   cd english_grammar_tutor

3. 安装依赖:
   pnpm install

4. 启动后端:
   pnpm dev:server

5. 启动应用:
   pnpm expo start --tunnel

详细说明请查看 QUICK_START.md
